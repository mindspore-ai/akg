# Email Writing Skill

专业邮件编写技能，帮助用户撰写各类场景的专业邮件。

## 适用场景

- 商务邮件
- 求职邮件
- 客户沟通
- 团队协作
- 投诉与反馈

## 邮件编写原则

### 1. 结构清晰

```
主题：简洁明确，直接说明邮件目的
问候：根据关系选择合适的称呼
正文：
  - 开场：说明写信目的
  - 主体：详细说明内容
  - 结尾：明确行动要求
签名：包含联系方式
```

### 2. 语气得体

- **正式场合**：使用"您"、"敬请"等礼貌用语
- **半正式**：保持专业但友好
- **非正式**：可以更加轻松但保持尊重

### 3. 内容精准

- 一句话说清楚目的
- 分段组织信息
- 使用列表突出要点
- 避免冗长和重复

## 常用模板

### 商务合作邮件

```
主题：关于[项目名称]合作事宜

尊敬的[姓名/职位]：

您好！

我是[公司名称]的[职位]，[简单自我介绍]。

我们注意到贵司在[领域]方面的专业实力，希望能探讨[具体合作内容]的可能性。

具体合作方向包括：
1. [合作点1]
2. [合作点2]
3. [合作点3]

如果您有兴趣，我们可以安排一次会面详细讨论。期待您的回复。

此致
敬礼

[您的姓名]
[职位]
[公司名称]
[联系电话]
[邮箱]
```

### 求职邮件

```
主题：应聘[职位名称] - [您的姓名]

尊敬的HR/招聘负责人：

您好！

我从[招聘渠道]获悉贵司正在招聘[职位名称]，我对该职位非常感兴趣。

我具备以下优势：
- [核心技能1]：[具体说明]
- [核心技能2]：[具体说明]
- [相关经验]：[具体说明]

我相信我的[特长]能为贵司的[业务/项目]带来价值。

随信附上我的简历，期待有机会与您详谈。

此致
敬礼

[您的姓名]
[联系电话]
[邮箱]

附件：[您的姓名]_简历.pdf
```

### 会议邀请邮件

```
主题：会议邀请 - [会议主题]

各位同事：

您好！

为了[会议目的]，我们计划召开一次会议，具体安排如下：

📅 时间：[日期] [时间]
📍 地点：[会议室/在线会议链接]
⏱ 时长：约[X]小时

会议议程：
1. [议题1] - [负责人]（[时长]）
2. [议题2] - [负责人]（[时长]）

- [准备事项1]
- [准备事项2]

请确认是否能参加，如有冲突请提前告知。

[您的姓名]
```

### 投诉/反馈邮件

```
主题：关于[产品/服务]的反馈

尊敬的客服团队：

您好！

我是贵司的客户，订单号为[订单号]。我想反馈以下问题：

问题描述：
[详细描述遇到的问题，包括时间、地点、具体情况]

期望解决方案：
[说明您希望如何解决]

相关凭证：
[如有，附上截图、订单号等]

希望能尽快得到处理，谢谢！

[您的姓名]
[联系方式]
```

## 邮件检查清单

发送前请检查：

- [ ] 主题是否简洁明确
- [ ] 称呼是否正确
- [ ] 目的是否在开头说明
- [ ] 内容是否分段清晰
- [ ] 是否有明确的行动要求
- [ ] 语气是否得体
- [ ] 是否有错别字
- [ ] 附件是否添加
- [ ] 收件人是否正确
- [ ] 必要时是否抄送相关人员

## 进阶技巧

### 1. 主题行优化

- ✅ 好：`关于Q1项目进度汇报 - 需要您审阅`
- ❌ 差：`项目汇报`

### 2. 突出重点

使用格式化突出关键信息：
- **加粗**重要内容
- 使用列表组织要点
- 用表情符号提示（✓ ⏳ 📅 ⚠️）

### 3. 行动导向

明确告知对方需要做什么：
- "请在本周五前确认"
- "需要您审批"
- "期待您的回复"

### 4. 礼貌用语

- 开头：您好、尊敬的
- 请求：麻烦、劳烦、恳请
- 感谢：感谢、谢谢、感激
- 结尾：此致敬礼、顺祝商祺

## 常见错误

1. **主题太模糊**："你好" "请教" "帮忙"
2. **直入主题不礼貌**：缺少问候直接提要求
3. **内容冗长**：一段到底，缺少分段
4. **缺少行动点**：说了一堆但不知道要对方做什么
5. **语气生硬**：缺少礼貌用语
6. **忘记附件**：正文提到但忘记添加
